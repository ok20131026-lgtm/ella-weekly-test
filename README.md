# Ella Weekly Test Maker

ESL Rainbows Orange Vol.2 위클리 테스트 예상문제 인터랙티브 앱입니다.

## 반영 기준
- 유닛 카드 클릭 시 해당 유닛 30문항 자동 시작
- Lesson 1 단어 5문항 + Lesson 2 단어 5문항
- Reading 1: 본문 직접형 7문항 + 변형형 3문항
- Reading 2: 본문 직접형 7문항 + 변형형 3문항
- 이전 문제 / 다음 문제 이동 가능

## 현재 완성 유닛
- Unit 1. Transportation: 30문항
- Unit 2. Wind: 30문항

## 잠금 유닛
- Unit 3~8은 교재 Lesson 1·2 본문 페이지가 추가로 필요합니다.

## 실행
```bash
npm install
npm run dev
```


## 최종 포함 범위

- Unit 1~Unit 8 전체
- 각 유닛 Lesson 1, Lesson 2만 포함
- Lesson 3 제외
- 각 유닛 30문항, 총 240문항
