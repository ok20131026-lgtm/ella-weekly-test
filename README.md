# Ella Weekly Test Maker

ESL Rainbows Orange Vol.2 Unit 1~8 위클리 테스트 예상문제 인터랙티브 앱입니다.

## 이번 수정 반영

- 단어 문제에 영영정의 한글뜻 힌트 표시
- 답 체크 후 모든 선지의 한글뜻 표시
- 정답 선지는 녹색 + `정답` 표시
- 사용자가 고른 오답 선지는 빨간색 + `오답` 표시
- 문제 시작 전에는 모든 선지가 흰색으로 표시
- 선택지 글자 크기 확대
- Reading 지문 글자 크기 확대
- 문제 화면 상단 홈 버튼 추가
- 홈으로 돌아가 다른 유닛 선택 가능

## 실행

```bash
npm install
npm run dev
```
