# Ella Weekly Test Maker

ESL Rainbows Orange Vol.2 Unit 1~8 위클리 테스트 예상문제 인터랙티브 앱입니다.

## 최종 반영 사항

- Unit 1~8 전체 포함
- 각 유닛 클릭 시 해당 유닛 30문항 자동 시작
- Lesson 1, Lesson 2만 포함
- Lesson 3 제외
- 단어 문제에 `힌트 보기` 버튼 추가
- 단어 문제에서 힌트 버튼을 눌렀을 때만 영영해석 표시
- 영영해석은 업로드 단어장 xlsm의 `영영해석` 열 기준 반영
- 리딩 문제에는 힌트/해석 미표시
- 답 체크 후 정답 선지 녹색 + `정답` 표시
- 답 체크 후 선택 오답 선지 빨간색 + `오답` 표시
- 단어 문제는 답 체크 후 모든 선지의 한글뜻 표시
- 문제 화면 상단 `홈` 버튼 추가
- 홈에서 다른 유닛 선택 가능
- 이전 문제 / 다음 문제 이동 가능

## 데이터 검수

- 전체 문항: 240문항
- 단어 문항: 80문항
- 단어 힌트: 80문항
- 리딩 힌트/해석: 0개

## 실행

```bash
npm install
npm run dev
```
