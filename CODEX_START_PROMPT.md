이 프로젝트는 ESL Rainbows Orange Vol.2 Unit 1~8 위클리 테스트 앱입니다.

유지해야 할 핵심 요구사항:
1. 메인 화면에는 Unit 1~8 카드가 표시되어야 한다.
2. 유닛 카드를 누르면 해당 유닛 30문항이 자동으로 시작되어야 한다.
3. 각 유닛은 Lesson 1, Lesson 2만 포함하고 Lesson 3은 제외한다.
4. 단어 문제에는 힌트 버튼을 표시한다.
5. 힌트 버튼을 눌렀을 때만 hintKo 값을 보여준다.
6. hintKo는 영영정의의 한글 해석이며, 단어장 xlsm의 영영해석 기준이다.
7. 리딩 문제에는 hintKo 또는 지문 해석을 표시하지 않는다.
8. 답 체크 후 정답 선지는 녹색 + 정답 라벨을 표시한다.
9. 답 체크 후 선택한 오답 선지는 빨간색 + 오답 라벨을 표시한다.
10. 단어 문제는 답 체크 후 모든 선지의 한글뜻을 표시한다.
11. 리딩 문제는 해석을 표시하지 않는다.
12. 문제 화면 상단에는 홈 버튼이 있어야 하며, 홈으로 돌아가 다른 유닛을 선택할 수 있어야 한다.
13. 이전 문제 / 다음 문제 이동 기능을 유지한다.
14. npm run build가 통과되도록 유지한다.
